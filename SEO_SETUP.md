# Настройка SEO для поисковых систем

## 1. Перезагрузка Nginx (если изменения не видны)

После загрузки файлов на сервер:

```bash
ssh root@<SERVER_IP>

# Перезагрузите Nginx
systemctl reload nginx

# Или используйте скрипт
chmod +x /tmp/reload-nginx.sh
/tmp/reload-nginx.sh
```

**Также очистите кеш браузера:**
- Chrome/Firefox: Ctrl+Shift+R (Windows) или Cmd+Shift+R (Mac)
- Или используйте режим инкогнито

## 2. Файлы для SEO уже созданы

✅ `robots.txt` - уже есть в проекте
✅ `sitemap.xml` - создан
✅ Мета-теги - добавлены в index.html

## 3. Загрузите sitemap.xml на сервер

```bash
scp public/sitemap.xml root@<SERVER_IP>:/var/www/elenafitmagic.ru/
```

## 4. Регистрация в поисковых системах

### Яндекс.Вебмастер

1. Перейдите на https://webmaster.yandex.ru/
2. Войдите через Яндекс аккаунт
3. Нажмите "Добавить сайт"
4. Введите: `elenafitmagic.ru`
5. Подтвердите владение сайтом одним из способов:
   - **HTML-файл** (рекомендуется):
     - Скачайте файл подтверждения
     - Загрузите на сервер: `scp файл.html root@<SERVER_IP>:/var/www/elenafitmagic.ru/`
   - **Мета-тег**:
     - Скопируйте мета-тег из Яндекс.Вебмастер
     - Добавьте в `<head>` файла `index.html`
     - Пересоберите и задеплойте
6. После подтверждения:
   - Перейдите в "Индексирование" → "Файлы Sitemap"
   - Добавьте: `https://elenafitmagic.ru/sitemap.xml`
   - Нажмите "Отправить на переобход"

### Google Search Console

1. Перейдите на https://search.google.com/search-console
2. Войдите через Google аккаунт
3. Нажмите "Добавить ресурс"
4. Выберите "Префикс URL" и введите: `https://elenafitmagic.ru`
5. Подтвердите владение сайтом:
   - **HTML-файл** (рекомендуется):
     - Скачайте файл подтверждения
     - Загрузите на сервер: `scp файл.html root@<SERVER_IP>:/var/www/elenafitmagic.ru/`
   - **Мета-тег**:
     - Скопируйте мета-тег из Google Search Console
     - Добавьте в `<head>` файла `index.html`
     - Пересоберите и задеплойте
6. После подтверждения:
   - Перейдите в "Sitemaps"
   - Добавьте: `sitemap.xml`
   - Нажмите "Отправить"

## 5. Дополнительные настройки SEO

### Проверка robots.txt

Убедитесь что файл доступен:
```bash
curl https://elenafitmagic.ru/robots.txt
```

Должен вернуть содержимое файла.

### Проверка sitemap.xml

```bash
curl https://elenafitmagic.ru/sitemap.xml
```

### Структурированные данные (Schema.org)

Можно добавить JSON-LD разметку для лучшего понимания сайта поисковиками. Добавьте в `index.html` перед `</head>`:

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "Елена Пильщикова",
  "jobTitle": "Персональный Фитнес-Тренер",
  "description": "Комплексное сопровождение для трансформации тела, здоровья и мышления",
  "url": "https://elenafitmagic.ru"
}
</script>
```

## 6. Время индексации

После регистрации в поисковых системах:
- **Яндекс:** обычно индексирует за 1-7 дней
- **Google:** обычно индексирует за 1-3 дня

Ускорить можно через "Запросить индексирование" в панелях вебмастеров.

## 7. Проверка индексации

Через несколько дней проверьте:

```bash
# Яндекс
site:elenafitmagic.ru

# Google
site:elenafitmagic.ru
```

## 8. Оптимизация контента

Для лучшего SEO убедитесь что:
- ✅ Уникальный контент на каждой странице
- ✅ Заголовки H1-H6 используются правильно
- ✅ Изображения имеют alt-атрибуты
- ✅ Внутренние ссылки между страницами
- ✅ Быстрая загрузка страниц
- ✅ Мобильная версия работает корректно
