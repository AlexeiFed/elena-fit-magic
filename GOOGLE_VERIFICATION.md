# Подтверждение владения сайтом в Google Search Console

## Что нужно сделать

Google требует добавить TXT запись в DNS для подтверждения владения доменом.

## Шаг 1: Скопируйте строку подтверждения

Из Google Search Console скопируйте строку:
```
google-site-verification=x6wG-0wreSgMevkJiPyb6vHb34_NAp6
```

## Шаг 2: Добавьте TXT запись в DNS Timeweb

1. Войдите в панель управления Timeweb
2. Перейдите: **Домены** → `elenafitmagic.ru` → **DNS**
3. Нажмите кнопку **"Добавить запись"** (фиолетовая кнопка справа)
4. Заполните форму:
   - **Тип:** TXT
   - **Хост:** `@` (или оставьте пустым для корневого домена)
   - **Значение:** `google-site-verification=x6wG-0wreSgMevkJiPyb6vHb34_NAp6`
   - **TTL:** 600 (или оставьте по умолчанию)
5. Нажмите **"Сохранить"** или **"Добавить"**

## Шаг 3: Подождите распространения DNS

DNS изменения могут занять:
- **Минимум:** 15-30 минут
- **Обычно:** 1-3 часа
- **Максимум:** до 24 часов

## Шаг 4: Проверка TXT записи

После добавления проверьте:

```bash
# Проверка TXT записи
dig elenafitmagic.ru TXT

# Или через онлайн сервисы:
# https://dnschecker.org/#TXT/elenafitmagic.ru
```

В выводе должна быть строка с `google-site-verification=...`

## Шаг 5: Подтверждение в Google Search Console

1. Вернитесь в Google Search Console
2. Нажмите кнопку **"ПОДТВЕРДИТЬ"** (темно-серая кнопка справа внизу)
3. Google проверит TXT запись в DNS
4. Если запись еще не распространилась, подождите и попробуйте снова через несколько часов

## Альтернатива: HTML-файл

Если TXT запись не работает, можно использовать HTML-файл:

1. В Google Search Console выберите способ подтверждения "HTML-файл"
2. Скачайте файл подтверждения
3. Загрузите его на сервер в корневую директорию сайта
4. Подтвердите владение

## После подтверждения

После успешного подтверждения:
1. Добавьте sitemap в Google Search Console:
   - Перейдите в **Sitemaps**
   - Добавьте: `sitemap.xml`
   - Нажмите **Отправить**

2. Запросите индексирование:
   - Перейдите в **Проверка URL**
   - Введите: `https://elenafitmagic.ru/`
   - Нажмите **Запросить индексирование**
