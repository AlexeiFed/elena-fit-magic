# Настройка Open Graph изображения

## Требования к изображению

Для корректного отображения превью в поисковой строке и соцсетях нужно изображение:

- **Размер:** 1200x630 пикселей (рекомендуется)
- **Формат:** JPG или PNG
- **Размер файла:** до 5 МБ (лучше до 1 МБ)
- **Соотношение сторон:** 1.91:1 (1200x630)

## Создание изображения

### Вариант 1: Использовать существующее изображение

Можно использовать одно из существующих изображений:
- `src/assets/elena-hero.jpg` - главное изображение
- `src/assets/elena-about.jpg` - изображение о Елене

Нужно обрезать/изменить размер до 1200x630px.

### Вариант 2: Создать новое изображение

Создайте изображение с:
- Фотографией Елены
- Названием сайта/услуг
- Контактной информацией (опционально)

## Инструкция по добавлению

1. **Создайте изображение 1200x630px** и сохраните как `og-image.jpg`

2. **Поместите файл в папку `public/`:**
   ```
   public/og-image.jpg
   ```

3. **Пересоберите проект:**
   ```bash
   npm run build
   ```

4. **Загрузите файл на сервер:**
   ```bash
   scp public/og-image.jpg root@<SERVER_IP>:/var/www/elenafitmagic.ru/
   ```

5. **Проверьте:**
   - Откройте https://elenafitmagic.ru
   - Используйте инструменты для проверки:
     - https://www.opengraph.xyz/url/https://elenafitmagic.ru
     - https://developers.facebook.com/tools/debug/
     - https://cards-dev.twitter.com/validator

## Альтернативные размеры

- **Facebook:** 1200x630px (рекомендуется)
- **Twitter:** 1200x675px (summary_large_image)
- **LinkedIn:** 1200x627px

Можно использовать одно изображение 1200x630px для всех платформ.

## Проверка после добавления

После загрузки файла проверьте мета-теги:

```bash
curl https://elenafitmagic.ru | grep -i "og:image"
```

Должно вернуть:
```
<meta property="og:image" content="https://elenafitmagic.ru/og-image.jpg" />
```
