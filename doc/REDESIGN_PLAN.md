|-------|----------|----------|
| `--primary` | `hsl(25 100% 50%)` | Яркий оранжевый |
| `--accent` | `hsl(25 95% 55%)` | Мягкий оранжевый |
| `--background` | `hsl(0 0% 98%)` | Почти белый |
| `--foreground` | `hsl(240 10% 3.9%)` | Почти чёрный |

### Выявленные проблемы

1. **Нет навигации** — сайт не имеет навбара. Пользователь не может быстро попасть к нужной секции. Единственная навигация — кнопка глобуса для языка.
2. **Однообразная сетка** — все секции используют одинаковый паттерн: заголовок по центру → grid карточек. Нет визуального ритма и контраста между секциями.
3. **WhyChoose перегружен** — 5 табов × (3 + 4 карточки) = ~35 карточек. Информационный шум.
4. **Нет социального доказательства** — для фитнес-тренера критично показать отзывы / трансформации клиентов.
5. **Карточки услуг одинаковые** — нет визуальной иерархии между Премиум и Базовым форматами.
6. **CTA в glass-панели** — ограниченное пространство снижает вес призыва к действию.
7. **Footer минимальный** — нет быстрой навигации, нет контактной информации кроме соцсетей.
8. **`App.css` содержит мёртвый код** — стили от Vite boilerplate (`.logo`, `.read-the-docs`) не используются.

## Принципы редизайна

- **Цветовая гамма неизменна** — orange primary, light background
- **Mobile-first** — каждая секция сначала проектируется под мобильное устройство
- **Минимум информационного шума** — убираем перегруженные секции, упрощаем
- **Social proof выше** — статы и отзывы ближе к началу страницы
- **Visual hierarchy** — premium выделен, есть чёткая точка входа
- **Используем design-tokens.ts** — все стили через единую систему
